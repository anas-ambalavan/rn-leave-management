# Leave Management App
> A leave management app using react native expo-cli
## Quick Start
``` bash
  # Install dependencies
  npm install
  
  # Run commands
  expo start
  Press a: open Android
  Press i: open iOS simulator
```  
## Debugging
```bash
  # start with this command 
  expo start --localhost
```
